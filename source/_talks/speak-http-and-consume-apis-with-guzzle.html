--- layout: talk title: Speak HTTP and Consume APIs With Guzzle level: Intermediate speakerlastname: Lindblom use: [talks, speakers] ---

Guzzle is an HTTP client library in PHP that puts all the power of cURL at your fingertips, but with a simple and intuitive interface. Learn how to use Guzzle to work with HTTP and web service APIs, and see how Guzzle's simple HTTP abstractions, flexible event system, and asynchronous request capabilities make Guzzle an amazing open source project that should be a permanent part of your PHP tool belt.
