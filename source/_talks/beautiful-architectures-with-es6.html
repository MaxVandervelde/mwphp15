ECMAScript 6, also known as JavaScript.Next, is ushering in a powerful new set of semantics for composing highly sophisticated real-time frontend applications. Learn the best practices for structuring your applications, as well as tools and libraries for writing and running tomorrow's JavaScript today, including React/Flux, Web Components, building blocks of the upcoming Angular 2.0.
