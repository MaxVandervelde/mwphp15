Yitzchok Willroth is an independent web application developer and software development consultant living in Lakewood, New Jersey. A start-up veteran, dev bootcamp instructor and frequent conference attendee, Yitzchok is the organizer of the ShorePHP User Group and an active participant in the PHPMentoring Initiative. Passionate about clean, expressive code and robust application architecture, he is also a strong proponent of remote teams as a way for software organizations to thrive while affording developers a healthy work:life balance.
